# Reviewer avatars

Drop one square/round photo per reviewer here, numbered to match their
position in the `reviewsData` array in main.js (1-indexed):

  1.jpg  -> reviewsData[0]  (currently "Julia S.")
  2.jpg  -> reviewsData[1]  (currently "EternalLord")
  3.jpg  -> reviewsData[2]  (currently "Dlaym")

If you reorder, add, or remove reviewers in reviewsData, the numbering
follows the new array order automatically — no other code changes needed.

Any reviewer without a matching file just keeps the plain "Avatar" circle
placeholder, so it's safe to leave some blank.
