# Coloring examples

Drop example images here as `1.jpg`, `2.jpg`, `3.jpg`, `4.jpg` (same
numbered pattern as every other commission folder).

These files are used in THREE places automatically, no extra upload needed:
- The "Coloring" card's cycling preview on the Commissions tab
- The gallery inside the Coloring config modal (click the card to open it)
- The "Coloring" tab on the Portfolio page (auto-pulled from here)

Missing numbers are fine — the site probes for files and only shows what
actually exists.
